# GLM52 SM121 B12x compiled backend

This module is the production C/CUDA runtime shell for the FlashInfer B12x fused MoE primitive. It does not contain a slow replacement kernel. It exports:

```text
SparkFlashInferB12xCompiledMoeCreate
SparkFlashInferB12xCompiledMoeLaunch
SparkFlashInferB12xCompiledMoeDestroy
```

It intentionally depends on the one-time generated B12x kernel table and exported TVM-FFI objects:

```text
SparkGlm52Sm121B12xGeneratedManifestInstance
SparkGlm52Sm121B12xGeneratedLaunch
```

Those symbols and objects are generated by `tools/glm52_b12x_aot_compile.py` on the SM121 machine and linked into the final decode-stage package. If they are absent, the package link fails.

Runtime does not import Python, Torch, FlashInfer, or CuTe DSL. It only calls the generated TVM-FFI object symbols through the generated C/CUDA launch table.
